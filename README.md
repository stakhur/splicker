# splicker
- python gui module (pyqt) which allow to move mouse pointer to specific location in several steps.
Initially - whole screen splitted by the grid. Each cell of the grid has an identificator (keyboard key).
If specific key is pressed - the cell with this key will be splitted by the new grid and mouse pointer moved into the center of the selected cell.
At any time Enter or Space can be pressed. It will emulate pressing of LMB or RMB respectevly.

The goal of the splicker is to make everything with the keyboard.
